Comprendiendo CSS Positioning

Posicionamiento Sticky 

          El posicionamiento sticky es un hibrido de los posicionamientos relative y fixed. El elemento
          es tratado como posicionado relative hasta que cruza un determinado umbral, en ese punto es
          tratado como posicionado fixed.

          Se debe especificar un valor para al menos una de las siguientes propiedades top, right, bottom
          o left, para que el posicionamiento sticky funcione como se espera.
