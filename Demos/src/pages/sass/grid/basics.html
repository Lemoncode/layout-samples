        Grid container -> The element on which display: grid is applied. It is the
        direct parent of all grid items.
      
        Grid item -> The children (direct descendants) of the grid container. The grand
        child items are not grid items.
      
        Grid Line -> The dividing lines that make up the structure of the grid. Can be
        either vertical (column grid lines) or horizontal (row grid lines) and reside
        on either side of a row or column.
      
        Grid Track -> The space between two adjancent grid lines. You can think of them
        like the colummns or rows of the grid.
      
        Grid Track -> The space between two adjancent grid lines. You can think of them
        like the colummns or rows of the grid.
      
        Grid cell -> The space between two adjancent row and two adjanecent column grid
        lines. It is a single unit of the grid.
      
        Grid Area -> The total space surrounde by four gid lines. A grid area may be
        comprised of any number of grid cells.
