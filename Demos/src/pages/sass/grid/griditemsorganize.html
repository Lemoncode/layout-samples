    justify-items -> Aligns the content inside a grid item along the column values
    axis (as opposed to align-items which aligns along the row axis). Applies
    to all grid items inside the container.

    This behavior can also be set on individual grid items via justify-self
    property.
  
    align-items -> aligns the content inside a grid item along the row axis (as
    opposes to justify-items which aligns the column axis). This value applies to
    all grid items inside the container.
